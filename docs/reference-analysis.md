# Reference Analysis

Studied for interaction principles only. No assets, copy, layouts, shaders, or code were copied. Each entry: what to borrow → what to avoid → conversion risk if copied too literally.

## BDSN (bdsn.club)
- **Hero:** restrained, type-led, work breathes on hover.
- **Borrow:** cursor-reactive project tiles; personality without clutter; alive-but-calm motion.
- **Avoid:** their exact palette/typeface; any literal layout.
- **Conversion risk:** if interactions hide the offer, real-estate buyers bounce. Keep CTA + pricing obvious.

## EPIC (epic.net/en)
- **Hero:** playful floating objects, configurator energy, memorable transitions.
- **Borrow:** floating "browser objects," strong micro-interactions, a transition system that feels intentional.
- **Avoid:** over-playfulness that reads as a toy, not a studio that ships business sites.
- **Conversion risk:** novelty for its own sake delays comprehension. Every playful beat must still advance the story (raw → launch).

## Unseen Studio (unseen.co/projects)
- **Hero/gallery:** spatial exploration, WebGL distortion, drag, infinite-world feel, optional sound.
- **Borrow:** spatial project gallery, smooth state-to-state transitions, "digital material" feel (liquid/elastic/magnetic).
- **Avoid:** replicating their shader look or drag-world wholesale; sound on by default.
- **Conversion risk:** an infinite-world with no exits/labels confuses buyers. Keep DOM text + clickable cards.

## Rive (rive.app)
- **Borrow:** premium typography, razor-sharp rendering, polished interaction states, very controlled color, lightweight interactive vector accents.
- **Avoid:** copying Rive's proprietary typeface; over-busy interface density.
- **Conversion risk:** none if kept restrained — this is the "trust" reference.

## Uncommon Design Group (uncommondesign.group)
- **Borrow:** clean structure, project-first storytelling, pinned showcase, large elegant type, strong whitespace, commercial info sitting calmly under the spectacle.
- **Avoid:** recreating their opening sequence too literally (explicitly called out).
- **Conversion risk:** low — this is the "clarity" reference. Lean on it for the sales half of the page.

## Active Theory — Frontier Within (activetheory.net)
- **Borrow:** cinematic pacing, living particle systems, procedural environments, visitor-controlled motion, "moving through an environment, not scrolling a document."
- **Avoid:** the biometric/body imagery and concept entirely — translate principles into Pilk.ai's "Living Website" world.
- **Conversion risk:** heavy full-screen immersion can wreck mobile + Lighthouse. Gate by device capability; ship a video/static fallback.

## Synthesis → Pilk.ai
Cinematic environment (Active Theory) + spatial gallery & material feel (Unseen) + restraint & cursor life (BDSN) + typographic precision (Rive) + commercial clarity (Uncommon) + memorable transitions (EPIC). The spectacle serves one job: make a visitor believe Pilk.ai can build them something extraordinary — then convert.
